🖍 기능
    
1. 로그인
   + 회원 가입

2. 회원 가입
   + 아이디 중복 체크
   + 비밀번호 / 비밀번호 확인

(메인 페이지) 
3. 거래 추가
   + 거래 내역 추가 (금액/수입,지출/카테고리/날짜)
     
4. 최근 거래 내역

5. 지출 내역 그래프
   + 카테고리 기반으로 얼마나 돈을 썼는지 그래프로 화면 표시
